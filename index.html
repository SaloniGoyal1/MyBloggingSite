What is Solar Power ?? Solar Power is the conversion of Sunlight into Energy so that we can use it. This conversion is done in many ways:- directly using Photovoltaics (PV), indirectly using Concentrated Solar Powers. Photovoltaics :- This system employs Solar Panels, each comprising a number of Solar Cells, which generate electrical power.

Ozone Layer or Ozone Shield or Ozonosphere, it is in the region of the upper Atmosphere. It is Present in Layer Stratosphere, from approximately 50 km (31 mi) above Earth. It absorbs most of the Sun’s Ultraviolet Radiations (mainly wavelength of UV Radiation is less than 310 nm). Ozone Layer Contain’s relatively high concentrations of Ozone molecules (O3).
